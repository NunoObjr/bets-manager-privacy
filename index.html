Política de Privacidade

O aplicativo Bets Manager respeita a privacidade dos usuários e não coleta, armazena ou compartilha dados pessoais.

1. Coleta de Informações

O Bets Manager não realiza coleta de dados pessoais. Todas as informações inseridas no aplicativo são armazenadas exclusivamente no dispositivo do usuário, de forma local.

2. Uso das Informações

Os dados inseridos pelo usuário são utilizados apenas para o funcionamento interno do aplicativo, como controle e organização de informações relacionadas às apostas.

3. Compartilhamento de Dados

O Bets Manager não compartilha dados com terceiros, não utiliza APIs externas e não realiza transmissão de dados pela internet.

4. Armazenamento e Segurança

As informações permanecem armazenadas localmente no dispositivo do usuário. O aplicativo não possui acesso a servidores externos.

5. Exclusão de Dados

O usuário pode excluir todos os dados a qualquer momento removendo o aplicativo do dispositivo ou utilizando as opções internas do app, quando disponíveis.
